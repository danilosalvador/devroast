# DevRoast 🔥

O DevRoast é uma aplicação web feita para desenvolvedores que não têm medo de ouvir a verdade. Cole seu código, escolha o nível de "ironia" e prepare-se para ser assado (roasted) pela inteligência artificial.

## ✨ Funcionalidades

- **Editor de Código Integrado**: Suporte a múltiplas linguagens com syntax highlighting premium (tema Vesper).
- **Roast Mode**: Alterne entre críticas brutais ou sarcasmo máximo.
- **Shame Leaderboard**: Veja os códigos mais "vergonhosos" da internet ranqueados pela comunidade.
- **Design Minimalista**: Interface dark mode focada no código, com estética industrial e limpa.

## 🚀 Tecnologias

Este projeto foi construído utilizando as tecnologias mais modernas do ecossistema web:

- **React 19** & **Vite**
- **Tailwind CSS v4** para estilização ultra-rápida.
- **Base UI** para componentes acessíveis e sem estilo base.
- **Shiki** para o melhor realce de sintaxe do mercado.
- **Biome** para um código limpo e organizado.
